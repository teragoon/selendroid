View project on GitHub

Hybrid/Web Elements

To switch to the web view mode:

Ruby: 

driver.switch_to.window('WEBVIEW')

Java: 

driver.switchTo().window("WEBVIEW");

Supported Element Locators

LocatorDescription
IdFinds the element by Id.
nameFind the element by name.
link textFind the element by text.
partial link textFind the element by partial text.
classFind the element by class.
tag nameFind the element by tag name.
cssFind the element by css locator.
xpathFind the element by xpath.

Id

Means the id that is described in the layout xml file of the corresponding activity.

Ruby: 

driver.find_element(:id,'header')

Java: 

driver.findElement(By.id("header"));

Name

Is mapped to the accessibility label (content description) of the view element.

Ruby: 

driver.find_element(:name,'name')

Java: 

driver.findElement(By.name("name"));

Link text

Mapped to the displayed text of the element.

Ruby: 

driver.find_element(:link_text,'Blog')

Java: 

driver.findElement(By.linkText("Blog"));

Partial Link Text

Mapped to the displayed text of the element.

Ruby: 

driver.find_element(:partial_link_text,'Blo')

Java: 

driver.findElement(By.partialLinkText("Blo"));

Class

Mapped to the ui element class of the view.

Ruby: 

driver.find_element(:class_name,'login')

Java: 

driver.findElement(By.className("login"));

tag name

Mapped to the simple name of the ui element class of the view.

Ruby: 

driver.find_element(:tag_name,'a')

Java: 

driver.findElement(By.tagName("a"));

css

Ruby: 

driver.find_element(:css,'ul.nav li')

Java: 

driver.findElement(By.cssSelector("ul.nav li"));

xpath

Ruby: 

driver.find_element(:xpath,'//a[@title='logo']')

Java: 

driver.findElement(By.xpath("//a[@title='logo']"));
